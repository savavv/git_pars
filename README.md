Автоматический тест для работы с GitHub API на Python

Требования
-Python 3.x
-Библиотека requests (pip install requests)


Установка
1) Клонируйте репозиторий: 
   -git clone https://github.com/savavv/git_pars
2) Перейдите в директорию проекта: 
   -cd git_pars
3) Установите зависимости из файла requirements.txt:
   -pip install -r requirements.txt

Настройка
1) Создайте файл .env в корне проекта и добавьте следующие переменные:
    GITHUB_USERNAME = Ваш логин github
    GITHUB_TOKEN = Ваш токен github
    REPO_NAME = Имя для тестового репозитория

Запуск теста
Выполните следующую команду для запуска теста:
-python test_api.py

Тест будет выполнять следующие действия:
-Создание нового публичного репозитория на GitHub с помощью API.
-Проверка списка репозиториев для подтверждения создания.
-Удаление созданного репозитория.
